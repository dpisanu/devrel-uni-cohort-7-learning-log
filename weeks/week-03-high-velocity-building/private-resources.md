# Private Resources: Week 03

## Purpose

This file documents which resources existed for Week 03 and how they were used without republishing private or cohort-only material.

## Public documentation boundary

This repository does not republish private DevRel University materials, including:

* Miro board screenshots
* full session transcripts
* recording files
* restricted cohort discussions
* private cohort links
* internal group chat details
* private screenshots from session material

## Notes on transcript use

The transcripts helped with accuracy, but the public documentation intentionally avoids exposing them. The published notes use short summaries, synthesized concepts and my own reflections rather than transcript excerpts.

## Notes on public links

Links to public projects or public articles may be included where useful, but the existence of a link in a private cohort context does not automatically make the surrounding discussion public.
