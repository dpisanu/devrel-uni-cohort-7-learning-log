# Assets

This folder contains visual and supporting assets used in this repository.

Assets are separated by ownership so that licensing and reuse permissions are clear.

## Folder structure

* `devrel-university/` — Assets that may be owned by DevRel University, its speakers, partners, contributors, or other rights holders.
* `own/` — Assets created by me for this learning log.

## Licensing

Assets in this folder may have different licensing terms depending on their source and ownership.

My original assets are generally licensed under the same license as my original written content in this repository, unless otherwise stated.

DevRel University-related assets are not covered by this repository’s Creative Commons license, MIT license, or any other license applied to my original work.

## Reuse

Before reusing any asset from this folder, check the README file in the relevant subfolder.

When in doubt, assume that third-party assets require permission from the relevant rights holder.
